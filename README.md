# Holberton School – Data Modeling & Persistence

Projects and exercises focused on **data modeling** and **persistence layers**, completed as part of the Holberton School curriculum.

This repository emphasizes designing schemas, working with relational databases, and implementing CRUD workflows.

## Repository structure

- `sql_crud/` – SQL CRUD exercises and practice queries

## Getting started

### Prerequisites

- A Unix-like environment (Linux/macOS/WSL recommended)
- SQL tooling depending on the project requirements (for example, a local database server and CLI client)

### Clone

```bash
git clone https://github.com/McPe4ce/holbertonschool-data_modeling_and_persistence.git
cd holbertonschool-data_modeling_and_persistence
```

## How to use this repository

1. Navigate into a topic directory (for example, `sql_crud`).
2. Read the module `README.md` for task descriptions and expected outputs.
3. Run the SQL scripts/commands using the database tooling required by the tasks.
